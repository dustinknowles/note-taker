# Note Taker
this is an app that lets a user type and save notes to their local machine.

## Installation
Run an 'npm i' in users CLI then 'npm start' to veiw locally on localhost:3000

## Technologies
- Node Js
- Express
- Heroku
- HTML
- CSS
